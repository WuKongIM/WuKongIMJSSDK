
## 文档

https://githubim.com/sdk/javascript


## npm 或yarn安装

```js
npm i wukongimjssdk
```
或者

```js
yarn add wukongimjssdk
```

## 在线体验

http://imdemo.githubim.com

## 构建

yarn build

## 发布

修改package.json里版本号

发布npm包

npm publish

## 引用

npm i wukongimjssdk



原生浏览器<script>的引入方式 使用lib/wukongimjssdk.umd.js 文件

原生引入需要加前缀例如：`wk.WKSDK.shared()`
